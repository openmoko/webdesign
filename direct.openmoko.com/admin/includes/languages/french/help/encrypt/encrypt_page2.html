A. Configuration:

The configuration only has 3 items that need to be setup, along with the key file.

1. Enable CC encryption Default: false
You can turn on and off the encryption, Once you turn on encryption the back off you can use the convert data screen to decrypt the data. If you do this it is advisable to remove the sensitive data. The order viewing and edit screen will decrypt the data so it is viewable.

2.Path to key File Default key/
Currently this is a directory within the includes/ directory.

3. Key File Name: cc_key.php
This is the file name for the key file, It is suggested that you use a different name then the one listed here. By clicking on this configuration setting you have three task to do.  b. Edit the key file name

Note: Once you have set the name of the file you can switch to Manage Key files
