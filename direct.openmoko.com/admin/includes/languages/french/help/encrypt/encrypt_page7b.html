A. Go to Encryption/Decryption side box click on Update CC Data . A list of orders with credit card number will apear. There will be a green message if the data is encrypted and a red message if it is not. Only the cc number is checked. You can decrypt either a single record or all records in this list

   Single record
    1. Click on the record not decrypted
    2. In the right hand box you will see the decrypt button, Click on this and the record will be decrypted.
    3. You will be retuned to the Order with cc numbers list. If there are more records go to step 2 and repeat until ther are not more red messages.
    4. You are now done if there are no more red messages

  All records
   1. Above the list of order with cc number you will see a button label decrypt . Click on this button.
   2. You are now done
