Web Developer

Openmoko is looking to immediately hire web developers to join our team. We are looking for hardcore server-side programmers. Not IT people. We use Open Source technologies on the server as well as our mobile devices. In general, we look for broad creative and technical skills, rather than the ability to program in one language over another. We concentrate on finding people with breadth, depth, communication skills and the ability to collaborate. If you have those attributes, we can teach you the specific tools.

Job location: Taipei, Taiwan

  • Requirements
  • Strong hands-on knowledge of Open Source web technologies
  • 5+ years programming experience
  • Ability to extend existing Open Source projects
  • Familiarity with and adherence to web standards
  • English language skills highly desired.
