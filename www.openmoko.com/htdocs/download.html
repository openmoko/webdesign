Openmoko Om 2008

Om 2008 is the first release of Openmoko's new software initiative. It is an X11 based distribution which uses EFL for the application launcher, custom UI apps and Qtopia on X11 for telephony. Check back for regular monthly updates as progress on the software continues. This is the first step from gtk on X11 to using multiple toolkits in combination. If you are not comfortable using software that is under development, please download the Qtopia distribution.
