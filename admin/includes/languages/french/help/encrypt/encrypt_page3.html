A. On opening the Manage Key Files screen it will look like the above screen with no key files.
Click on the CREATE files button and a empty key file will be set up using the name you selected in the text input box.

B. You will now have two empty key files. Starting off you will only need to enter a key phrase into the Primary key file. Next click on the EDIT button next to the Primary key file. This will open the key file edit screen.

C. To edit the "Primary" Click on the EDIT button at the end of the row. A Key Edit screen will open.

Note: The new key file is used to allow updating the data encryption when you want to change to a new encryption key.

D. Type in your key phrase in the text box and click the SAVE button. This will save the phrase. The phase can be anything from random numbers and letters to a phrase you can remember.

E. From this point on all cc information will be encrypted as it is entered into the orders by the customer. The next step you may need to take might be to encrypt existng Credit card information.

F. Next click on the Update cc data in the lefthand side box.
